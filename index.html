Stock Newsletter Alert
Odoo Module Inventory
Overview

An Odoo inventory module that monitors product stock levels and automatically triggers branded email notifications to newsletter subscribers. When stock drops below a configurable threshold, an alert goes out. When it's restocked, a follow-up "back in stock" email fires automatically — all without any manual intervention. Built using Odoo's scheduled actions (cron) and the native mail system for reliable delivery.

Key Features
Configurable low-stock threshold per product
Automated "low stock" alert email to subscribers
"Back in stock" notification on restock
Branded QWeb email template
Scheduled via Odoo cron (no manual trigger needed)
Subscriber list managed via Odoo newsletter contacts
Tech Stack
Python XML QWeb Email Stock Module Scheduled Actions Mail Module

Website Quotation System
Odoo Module Website
Overview

A website-facing Odoo module that lets customers request a price quote directly from a product page on the Odoo website. The request creates a draft quotation in the Odoo Sales module, sends the customer a clean branded PDF estimate, and notifies the sales team — all automatically. The feature can be enabled or disabled from backend settings, making it perfect for businesses that operate quote-first rather than showing public prices.

Key Features
"Request a Quote" button on product pages
Auto-creates draft Sale Order in backend
Branded PDF quotation emailed to customer
Sales team notified via Odoo activity
Toggle on/off from Website settings
Custom QWeb report for PDF design

Website Hide Price
Odoo Module eCommerce
Overview

A lightweight but highly practical Odoo eCommerce module that gives store admins full control over price visibility on the website. With a single toggle in Website Settings, all product prices can be hidden from public/portal visitors based on configuration. Perfect for B2B stores, tender-based businesses, or any store that doesn't want to show pricing publicly before a customer registers or logs in.

Key Features
Single toggle in Website Settings
Hides prices on product list & product pages
Hides prices on cart and checkout
Hides price filter for products
Works with Odoo's multi-website setup
Maintain price visibility for public and portal users separately
Tech Stack
Python XML OWL / JS Website Settings eCommerce QWeb
